# نظام حجز المواعيد الطبية

نظام متكامل لحجز المواعيد الطبية مع لوحات تحكم للأطباء والمرضى والإدارة.

## التشغيل السريع

```bash
# تشغيل النظام
RUN.bat

# إيقاف النظام
stop.bat
```

## الروابط

- Frontend: http://localhost:3000
- Backend: http://127.0.0.1:8090/_/
